# ersha-prime

The main component of the ersha project and is responsbile to manage the [`edge devices`](../ersha-edge) and the [`dispatcher registry`](../ersha-dispatch).
It can be accessed through RPC and API resquest.

## Usage

```
cargo run -p ersha-prime ./config-file.toml
```

**NB:** A sample configuration file is accessible through [`ersha-prime.toml`](ersha-prime.toml).
